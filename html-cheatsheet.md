# HTML Cheat sheet

```<hr>``` : horizontal rule

```<ul>``` : unordered list

```<ol>``` : ordered list

```<li>``` : list item

---

Unordered list Sample

```html
<ul>
    <li>First Item</li>
    <li>Second Item</li>
</ul>
```

---

Ordered list Sample

```html
<ol>
    <li>First Item</li>
    <li>Second Item</li>
</ol>
```

---

```<span>``` : Like a div but it's an Inline element instead of a block element.
